# Classical Lullabies MIDI Collection

Pieces that are either explicitly lullabies or have a lullaby-like character —
gentle, rocking, dreamlike, or tender enough to soothe a child to sleep.

## How to download

Run the download script from the project root:
```bash
./download_lullabies.sh
```

For any files that fail to download automatically, the script will list
manual download sources (kunstderfuge.com, musescore.com, bitmidi.com,
piano-midi.de, mfiles.co.uk).
